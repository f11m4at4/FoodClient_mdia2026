# 역할
너는 10년차 안드로이드 전문 개발자야.
프로젝트 구조화 및 설계에 경험이 많아.
AI 활용 프로젝트 제작 경험이 다수 있어.

## 배경
https://github.com/facebook/meta-wearables-dat-android/tree/main/samples/CameraAccess
이 링크는 meta ray-ban CameraAccess 샘플이야.
이 샘플을 이용해서 사용자가 음성으로 "사진찍어줘"라고 요청해 음식 사진을 찍으면
OpenAI API를 통해 이미지를 인식해서 어떤 음식인지 파악하고 조리법을 500자 이내로 스마트폰에 텍스트 UI 로 표시하고
TTS로 음성으로 설명하는 앱을 만들려고해.

## 요구사항
다음 내용이 반영 되도록 계획을 세워서 `docs/PLAN.md`에 작성해줘.
- STT로 "사진찍어줘"라고 말하면 카메라 버튼을 누른 것과 동일하게 사진이 촬영되도록 해줘.
- STT로 인식된 문구는 화면 UI에 표시.
- 스트리밍 중에는 계속 리스닝하고 있어야 하고(TTS 재생 중엔 잠깐 멈췄다가 재개)
- 음식 인식은 앱이 OpenAI API를 직접 호출하지 말고 분석 서버를 경유해줘.
- 분석 서버 접속은 로컬 개발용 사설 IP `http://`로 접속해줘.

### 조건
- 테스크 작성은 mock device를 기반으로 먼저 작업하고 이 후 스마트글래스 테스트 작업으로 확장한다.
