# 배경
`docs/PLAN.md`에 프로젝트 계획이 작성되어 있다.

# 요구사항
`docs/PLAN.md`의 계획을 `docs/tasks`에 Task 단위로 분리해서 md 파일로 만들어줘.
- 안드로이드에서 촬영한 음식 사진을 분석 서버로 전송해 결과를 받는 기능 구현시, 캡처 성공과 분석 요청 실패를 분리하고 logcat 확인, 사용자 오류 메시지 처리를 고려해서 제작해.
- `docs/checklist` 폴더에 TASK 별 체크리스트가 작성된 MD 파일을 만들어줘.
- task는 이해하기 쉽게 작성하고 설계가 복잡해 지지않게 단계를 너무 많이 나누지 않도록해.
