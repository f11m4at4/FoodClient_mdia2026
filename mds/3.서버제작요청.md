# 역할
너는 10년차 Python 서버 전문 개발자야.

# 요구사항
분석 서버 제작을 위한 프롬프트를 docs 폴더에 `FoodServer.md` 파일로 만들어줘.
- Python FastAPI를 이용하도록해.
- OpenAI API를 사용하고 모델은 gpt-4o-mini를 사용해.
- 구현된 분석서버 통신 내용을 바탕으로 요청과, 응답에서 사용할 json 데이터 예시를 작성해줘.
- 분석 서버 주소는 `local.properties`에 `ANALYZE_SERVER_URL` 변수에 저장해서 사용하도록 해.
